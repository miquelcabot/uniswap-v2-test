# uniswap-test
Test app that connects to Uniswap using Hard Hat + Waffle + Ethers

Install:
```
npm install
```

Compile:
```
npm run compile
```

Test in local Hardhat blockchain:
```
npm run test
```

Test in Rinkeby test network:
```
npm run test_rinkeby
```

Run prettier on all your contracts:
```
npm run prettier
```
