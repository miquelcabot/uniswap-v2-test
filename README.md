# uniswap-test
Test app that connects to Uniswap using Hard Hat + Waffle + Ethers

Install:
```
npm install
```

Compile:
```
npm run compile
```

Test:
```
npm run test
```

Run prettier on all your contracts:
```
npm run prettier
```
